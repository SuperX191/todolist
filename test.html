ผักและผลไม้

ประโยชน์ของผักและผลไม้แต่ละชนิด วิตามินและแร่ธาตุที่สำคัญ รวมถึงวิธีรับประทานให้ได้ประโยชน์สูงสุด

สูตรอาหารสุขภาพ

สูตรอาหารเพื่อสุขภาพที่ทำได้ง่ายๆ ที่บ้าน รสชาติอร่อยและดีต่อสุขภาพ เหมาะสำหรับทุกคนในครอบครัว

อ่านเพิ่มเติม

คำแนะนำการกินผัก

ผักใบเขียว

  • ผักโขม - อุดมด้วยธาตุเหล็กและแคลเซียม
  • คะน้า - มีวิตามินเอและซีสูง
  • ผักกาดหอม - แคลอรี่ต่ำ มีไฟเบอร์สูง
  • บรอกโคลี - มีสารต้านอนุมูลอิสระสูง

ผักสีส้มและเหลือง

  • แครอท - อุดมด้วยเบต้าแคโรทีน
  • ฟักทอง - มีวิตามินเอสูง ดีต่อสายตา
  • พริกหวานเหลือง - มีวิตามินซีสูง
  • ข้าวโพด - มีไฟเบอร์และสารต้านอนุมูลอิสระ

ผักสีแดงและม่วง

  • มะเขือเทศ - มีไลโคปีนป้องกันมะเร็ง
  • บีทรูท - ช่วยลดความดันโลหิต
  • กะหล่ำปลีม่วง - มีสารแอนโทไซยานิน
  • พริกแดง - มีแคปไซซินช่วยเผาผลาญไขมัน

ท่าออกกำลังกายพื้นฐาน

สควอช (Squat)

ท่าออกกำลังกายพื้นฐานสำหรับกล้ามเนื้อขาและสะโพก

  1. ยืนแยกขากว้างเท่าช่วงไหล่
  2. ย่อตัวลงเหมือนจะนั่งเก้าอี้ หลังตรง
  3. ย่อจนต้นขาขนานกับพื้น แล้วค่อยลุกขึ้น
  4. ทำ 3 เซต เซตละ 12-15 ครั้ง

วิดพื้น (Push-up)

ท่าออกกำลังกายสำหรับกล้ามเนื้อหน้าอกและแขน

  1. นอนคว่ำ วางมือกว้างเท่าช่วงไหล่
  2. ดันตัวขึ้นจนแขนตึง หลังตรง
  3. ค่อยๆ ลดตัวลงจนอกเกือบแตะพื้น
  4. ทำ 3 เซต เซตละ 10-12 ครั้ง

ครันช์ (Crunch)

ท่าออกกำลังกายสำหรับกล้ามเนื้อหน้าท้อง

  1. นอนหงาย งอเข่า วางเท้าราบกับพื้น
  2. วางมือประสานที่ท้ายทอย
  3. ยกศีรษะและไหล่ขึ้นจากพื้น
  4. ทำ 3 เซต เซตละ 15-20 ครั้ง
